# PHQ-9 Screening Tool

A PHQ-9 educational screening tool with:

- An interactive Python command-line app
- A Vercel-ready static web app

## Run The CLI

```bash
python phq9_screening_tool.py
```

## Preview The Web App

Open `index.html` in a browser, or run:

```bash
npm start
```

## Deploy To Vercel

1. Push this repository to GitHub.
2. Import it in Vercel.
3. Keep the default static project settings.
4. Deploy.

The browser app is served from `index.html` and does not require a build step.

## Important

This tool is for educational and informational screening purposes only. It is not a diagnostic tool and does not replace professional medical advice, diagnosis, or treatment.

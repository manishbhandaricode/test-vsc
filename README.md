# SIP Calculator

A simple, responsive SIP calculator built with HTML, CSS, and vanilla JavaScript.

## Local Preview

Open `index.html` directly in a browser, or use Vercel's local dev server:

```bash
npm install
npm start
```

## Deploy To Vercel

1. Push this repository to GitHub.
2. Import the repository in Vercel.
3. Keep the default static project settings.
4. Deploy.

The app uses `index.html` as the static entry point and does not require a build step.

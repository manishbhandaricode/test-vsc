# Tapas Ruling Atlas by Manish

A Vercel-ready rebuilt Tapas Ruling Atlas by Manish workspace, bundled with the local ruling dataset for static deployment.

## Features

- Local rulings catalogue loaded from `assets/data/ruilings.json`
- Keyword search, category, sub-category, stage, court, and sort filters
- Refined result cards with pagination, concise previews, and detail-count signals
- Full issue, holding/ratio, category, sub-category, stage, court, statutory anchors, practice notes, related details, web references, and data provenance preserved in the detail modal
- Light/dark theme switch with local preference storage
- Custom "M" favicon mark
- Add, edit, delete, copy citation, reset, and main website controls
- Keyboard-accessible cards and search relevance ranking
- Local browser persistence for edits, custom rulings, and locally deleted bundled entries
- Static deployment on Vercel with no build step

## Run

Serve the folder with any static server, or use:

```bash
npm start
```

## Deploy

Import this repository into Vercel and keep the default static project settings.
